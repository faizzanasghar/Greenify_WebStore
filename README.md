# Plant Shop E-Commerce Website

A responsive web-based e-commerce platform for purchasing plants, seeds, and gardening accessories.

## 📋 Project Description

This is a full-featured plant shop website that allows users to browse and purchase various categories of plants and gardening products. The platform includes user authentication, product catalog, shopping cart functionality, and customer support features.

## ✨ Features

- **User Authentication**: Login system for customer accounts
- **Product Categories**: 
  - Indoor Plants
  - Flower Plants
  - Climber Plants
  - Vegetable Seeds
  - Decorations & Artificial Plants
  - Plants Accessories
- **Shopping Cart**: Add and manage items
- **Product Browsing**: Detailed product pages for each category
- **Customer Support**: FAQ and Contact pages
- **Blog Section**: Information and gardening tips
- **About Page**: Company information
- **Responsive Design**: Works on desktop and mobile devices

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Structure**: Semantic HTML
- **Styling**: Custom CSS
- **Interactivity**: Vanilla JavaScript

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side setup required for basic functionality

### Installation

1. Clone or download the project files
2. Open `index.html` in your web browser to start browsing
3. No additional dependencies or installations needed

## 📖 Usage

1. **Browse Products**: Navigate through different plant categories using the menu
2. **View Details**: Click on products to see more information
3. **Add to Cart**: Click "Add to Cart" button to purchase items
4. **Login**: Create an account or log in using the login page
5. **Contact**: Use the contact page to reach customer support
6. **FAQ**: Visit FAQ for common questions
